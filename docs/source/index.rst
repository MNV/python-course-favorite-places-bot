Favorite Places Bot
==============

Чат-бот предназначен для сохранения любимых мест и получения справочной информации
по ним (страна, город, погода и курсы валют).

Функции чат-бота
----------------

Чат-бот для Telegram по запросу пользователя сохраняет его текущее местоположение,
передавая информацию в соответствующий микросервис для обогащения данных
информацией о стране, городе, погоде и курсах валют.

Пользователь может просмотреть список ранее сохраненных любимых мест с дополнительной информацией,
редактировать описание любимых мест и удалять места.

В чат-боте предусмотрен интерфейс с кнопочным меню для возможности взаимодействия с пользователем.

Технологический стек
--------------------

- Docker
- Python 3
- Sphinx
